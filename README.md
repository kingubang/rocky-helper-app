# 洛克百宝箱 - 安装与使用指南

## 项目概述
洛克王国世界手游悬浮窗辅助查询工具（Android APP），可在游戏过程中以悬浮窗形式随时查询精灵图鉴、属性克制关系、培养推荐。

## 如何编译安装

### 方式一：使用 Android Studio（推荐）

1. 安装 [Android Studio](https://developer.android.com/studio)
2. 打开 Android Studio → File → Open → 选择 `rocky-helper-app` 文件夹
3. 等待 Gradle 同步完成（首次可能需要下载依赖）
4. 连接你的小米16 Pro Max（开启开发者选项 + USB调试）
5. 点击工具栏的 Run (绿色三角形) 按钮

### 方式二：命令行编译

1. 确保已安装 JDK 17+ 和 Android SDK
2. 在项目根目录执行：
   ```
   gradlew.bat assembleDebug
   ```
3. 生成的 APK 位于：`app/build/outputs/apk/debug/app-debug.apk`

## 首次使用

1. 安装 APP 后打开「洛克百宝箱」
2. 点击「开启悬浮窗」按钮
3. 授予悬浮窗权限（小米手机需在 设置 → 应用管理 → 洛克百宝箱 → 权限管理 → 悬浮窗 中手动开启）
4. 授予通知权限
5. 悬浮球🔮会出现在屏幕上
6. 点击悬浮球打开查询面板
7. 切换到洛克王国世界，悬浮球会覆盖在游戏画面上
8. 拖动悬浮球可移动位置，松手会自动靠边吸附

## 功能说明

| Tab | 功能 |
|-----|------|
| 🏠 首页 | 功能介绍和使用说明 |
| 📖 图鉴 | 搜索精灵名称/编号，按属性筛选，查看种族值、进化链、弱点、培养方案 |
| ⚔️ 克制 | 18属性克制矩阵，支持防御端弱点/抗性查询 |
| ⭐ 推荐 | T0/T1精灵强度榜和培养建议 |

## 注意事项

- 本工具仅提供信息查询功能，不含任何自动化操作，完全合规安全
- 悬浮窗点击内容区域后可输入搜索，点击标题栏后返回不可聚焦状态（不影响游戏）
- 游戏运行时建议将悬浮球拖到屏幕右侧边缘，避免遮挡操作
